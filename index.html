📊 Oseti準拠 日本語感情分析ツールOseti-based Japanese Sentiment Analyzer

🎯 このツールについてAbout This Tool

このツールは、日本語テキストの感情極性（ポジティブ/ネガティブ）を分析するWebベースのツールです。Osetiライブラリ（池上有希乃氏開発のPython用感情分析ライブラリ）の考え方に準拠し、東北大学の日本語評価極性辞書を使用しています。

This is a web-based tool for analyzing sentiment polarity (positive/negative) of Japanese text, based on the Oseti library approach (a Python sentiment analysis library developed by Yukino Ikegami) and using the Japanese Sentiment Polarity Dictionary from Tohoku University.

📌 使い方: ① 下の「📖 完全版辞書を読み込む」をクリック → ② 表示される入力欄に日本語テキストを入力 → ③ 「🚀 分析開始」をクリック 📌 How to use: ① Click "📖 Load the Full Dictionary" below → ② Enter Japanese text in the box that appears → ③ Click "🚀 Start Analysis"

主な機能:Key Features:

  • 各単語に-1.0（最もネガティブ）から+1.0（最もポジティブ）のスコアを付与
  • テキスト全体の感情スコアを計算し、総合的な感情を判定
  • Kuromoji形態素解析器（IPAdic辞書使用）による単語分割
  • 文ごとの分析に対応し、感情の変化を追跡可能
  • ✨ 後方否定・並列否定検出（拡張機能）
  • レビュー分析、SNS分析、顧客フィードバック分析などに活用可能
  • Assigns scores from -1.0 (most negative) to +1.0 (most positive) to each word
  • Calculates an overall sentiment score and determines the general sentiment
  • Word segmentation using the Kuromoji morphological analyzer (with the IPAdic dictionary)
  • Supports sentence-by-sentence analysis to track sentiment changes
  • ✨ Backward & parallel negation detection (extended feature)
  • Useful for review analysis, social media analysis, customer feedback analysis, etc.

⚠️ 「拡張機能(後方否定・並列否定検出)」とは? — クリックで説明を表示What are the "extended features (backward & parallel negation detection)"? — click to expand

本ツールは元のosetiライブラリに加えて、以下の機能を追加実装しています: This tool implements the following features in addition to the original oseti library:

【否定検出の拡張】Extended Negation Detection
元のoseti: 否定語(ない・ず・ぬ等)が現れたとき、直前の感情語1語のみを反転(例:「楽しくない」→ 検出)
本ツール: 助詞を挟んだ否定や、複数語にかかる否定も検出可能(下記トグルでON/OFF可能)
対応例: 「お金がない」「希望はない」「期待できない」「満足できない」 Original oseti: flips only the single emotion word immediately preceding a negation word (ない, ず, ぬ, etc.) — e.g. 「楽しくない」 is detected
This tool: also detects negation across particles and negation affecting multiple words (toggle below)
Examples: 「お金がない」「希望はない」「期待できない」「満足できない」
【並列否定】Parallel Negation
複数の感情語が並列助詞で繋がれた否定表現にも対応しています。
対応例: 「お金も希望もない」「雰囲気も料理も良くない」「サービスも期待できない」「値段も配送も満足できない」 Supports negation patterns with multiple emotion words connected by parallel particles.
Examples: 「お金も希望もない」「雰囲気も料理も良くない」「サービスも期待できない」「値段も配送も満足できない」

研究で使用する際は、この拡張機能（後方否定・並列否定検出）を使用したことを論文に明記してください。 When using in research, please specify in your paper that these extended features (backward & parallel negation detection) were used.

文を辞書と文法にもとづいて単語に区切る処理です。ON: 正確な単語分割(推奨) / OFF: 文字種の切り替わりで区切る簡易分割(精度が下がります。Kuromojiを読み込めない場合の代替) Morphological analysis splits sentences into words using a dictionary and grammar. ON: accurate segmentation (recommended) / OFF: simple segmentation by character type (less accurate; fallback when Kuromoji cannot be loaded)
ON: 助詞を挟んだ否定(「期待できない」)や並列否定(「お金も希望もない」)も検出 / OFF: 元のosetiと同じ互換モード(否定語の直前の感情語1語のみ反転)。詳しくは上の「拡張機能とは?」を開いてください ON: also detects negation across particles (「期待できない」) and parallel negation (「お金も希望もない」) / OFF: original-oseti-compatible mode (flips only the single emotion word immediately preceding a negation word). See "What are the extended features?" above for details

The "backward & parallel negation detection" features are original extensions not included in the original oseti library. See the explanation in the "About Extended Features" section above. When using in research, please specify that these extended features were used.
